# VideoRecorder
#### 一个视频录像机项目

- [x] 高性能任意尺寸视频录制 (以相机支持的最大尺寸预览)

- [X] 断点录制 (断点回删)

- [X] 离屏录制

- [X] 录制时的Canvas API支持

- [ ] 实时滤镜

- [ ] 后期滤镜
